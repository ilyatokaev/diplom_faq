Установка и первый запук

  • Скопировать содержимое этой папки, включая все подпопаки, в целевое место расположения системы
  • На сервере, на котром будет работать система, в MySQL необходимо создать новую БД.
  • В созданой БД необходимо запустить SQL-скрипт faq.sql, который создаст необходимые объекты в БД и заполнит БД первичными данными
  • В целевой папке в подпапке Classes/ необходимо отредактировать файл Cfg.php. А именно в статическом методе getDB() в строке $conn = new PDO(.....) необходимо прописать строку подключения к созданой БД".
  • Для запуска системы необходимо в браузере указать маршрут к файлу index.php в контексте созданного сайта.
